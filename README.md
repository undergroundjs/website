## Getting Started

### Prerequisites

- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/)

1. To get started copy the repo:

```
git clone <Repo URL>.git
```

2. Install the project packages:

```
npm install
```

3. Run the server:

```
npm start
```
